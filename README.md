# Store Rating App

## 📌 Project Overview

The Store Rating App is a full-stack web application that allows users to browse and rate stores, while administrators can manage stores, users, and ratings through a secure dashboard. The application provides role-based access and a user-friendly interface for managing store information and feedback.

## 🚀 Features

### User

* Register and log in securely
* View the list of stores
* Search and filter stores
* Submit and update store ratings
* View personal rating history

### Store Owner

* View ratings received for their store
* Monitor average store rating
* Access store-specific dashboard

### Admin

* Add, edit, and delete stores
* Manage users and store owners
* View all ratings
* Monitor application statistics
* Control platform data through an admin panel
